# INDOS Number Online Application Guide 2026

**Your INDOS (Indian National Database of Seafarers) number is your permanent identity as an Indian seafarer.** Without it, you cannot apply for CDC, appear for DG Shipping exams, or join any vessel. This guide walks you through the complete online application process.

## What is INDOS Number?

INDOS is the centralized database maintained by DG Shipping that tracks every registered Indian seafarer. Your INDOS number:

- Remains with you throughout your maritime career
- Is required for all DG Shipping transactions
- Links your CDC, COC, and other certificates
- Is mandatory for joining any merchant vessel

**Important**: One person can have only ONE INDOS number. If you've already registered, do not create duplicate entries.

## Documents Required for INDOS Application

### Mandatory Documents (for all applicants)

1. **Passport** (valid for minimum 6 months)
   - Front and back page scan
   - All stamped pages if applicable

2. **Educational Certificates**
   - 10th Mark sheet and Certificate
   - 12th Mark sheet and Certificate (if applicable)
   - Degree certificates (if applicable)

3. **Birth Certificate or 10th Certificate** (for date of birth proof)

4. **Aadhaar Card** (front and back)

5. **Passport-size Photograph**
   - White background
   - Recent (within 3 months)
   - Size: 3.5 cm x 4.5 cm
   - Format: JPEG, max 100 KB

### Additional Documents (based on category)

**For Pre-sea course admission:**
- Course admission letter
- Institute approval letter from DG Shipping

**For existing seafarers:**
- CDC (if available)
- COC (if available)
- Sea service documents

## Step-by-Step Online Application Process

### Step 1: Access DG Shipping Portal

1. Visit [https://dgshipping.gov.in](https://dgshipping.gov.in)
2. Click on "e-Governance" section
3. Select "INDOS Registration"
4. Click "New Registration"

### Step 2: Create Account

1. Enter valid email address
2. Enter mobile number
3. Create strong password
4. Complete captcha
5. Click "Register"
6. Verify OTP sent to mobile and email

### Step 3: Fill Personal Details

**Section A - Basic Information:**
- Full name (exactly as in passport)
- Date of birth
- Gender
- Nationality (Indian)
- Father's name
- Mother's name

**Section B - Contact Details:**
- Permanent address
- Present address
- Mobile number
- Emergency contact

**Section C - Physical Details:**
- Height (in cm)
- Identifying marks
- Blood group

### Step 4: Educational Qualifications

Enter details of all educational qualifications:
- Board/University name
- Year of passing
- Percentage/CGPA
- Subjects studied
- Certificate number

### Step 5: Upload Documents

Upload all required documents in specified format:
- **Format**: PDF or JPEG
- **Size**: Maximum 500 KB per document
- **Quality**: Clear, readable scans

**Pro tip**: Use a mobile scanning app like CamScanner for clear document scans.

### Step 6: Select Pre-sea Course (if applicable)

If applying for INDOS before joining a pre-sea course:
- Select course type (DNS, GME, GP Rating, etc.)
- Select approved institute
- Enter admission date

## How to Check INDOS Application Status

1. Login to DG Shipping portal
2. Go to "My Applications"
3. View status of your INDOS application

Status meanings:
- **Submitted**: Application received
- **Under Verification**: Documents being checked
- **Query Raised**: Additional information needed
- **Approved**: INDOS number allotted
- **Rejected**: Application denied (reasons provided)

## Common Mistakes to Avoid

### 1. Name Mismatch
Your name must match EXACTLY across all documents. Even small variations like "KUMAR" vs "Kumar" can cause rejection.

**Solution**: Use passport name as standard for all entries.

### 2. Poor Quality Scans
Unclear or blurry document scans lead to rejection.

**Solution**: Use proper scanning apps, ensure good lighting, check readability before upload.

### 3. Wrong Document Format
Uploading documents in wrong format causes errors.

**Solution**: Convert all documents to PDF. Keep file size under 500 KB.

### 4. Incomplete Information
Leaving fields blank or entering "NA" incorrectly.

**Solution**: Fill all mandatory fields. If something doesn't apply, check if the field is actually mandatory.

### 5. Multiple Applications
Creating multiple applications for same person.

**Solution**: One person should have only ONE INDOS number. If previous application exists, track it instead of creating new.

## What to Do If INDOS Application is Rejected?

1. **Read rejection reason carefully** - DG Shipping provides specific reasons
2. **Correct the issues** - Usually document quality or information mismatch
3. **Reapply** - Submit fresh application with corrections
4. **Fee is non-refundable** - You'll need to pay again for reapplication

Common rejection reasons:
- Name mismatch between documents
- Unclear photographs
- Invalid/expired documents
- Incomplete information
- Duplicate application detected

## Frequently Asked Questions

**Q: Can I apply for INDOS without getting admission in pre-sea course?**
A: Yes, but you need to specify your intended course and institute.

**Q: My INDOS application has been pending for 30+ days. What should I do?**
A: Contact DG Shipping helpdesk or visit nearest MMD office with application reference number.

**Q: Can I change details in INDOS after registration?**
A: Yes, minor corrections can be made through the portal. Major changes require MMD visit with supporting documents.

**Q: I forgot my INDOS number. How can I retrieve it?**
A: Login to DG Shipping portal with registered email/mobile or contact MMD with your CDC number.

**Q: Is INDOS number same as CDC number?**
A: No. INDOS is database registration number. CDC is your seaman's identity document. Both are linked but different.
