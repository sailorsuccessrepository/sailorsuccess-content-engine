# INDOS Registration Guide 2026: Step-by-Step Process

INDOS (Indian National Database of Seafarers) number is your unique identification as an Indian seafarer. This 8-digit number is mandatory for CDC application and all maritime activities.

## What is INDOS?

**INDOS = Indian National Database of Seafarers**

- Maintained by Directorate General of Shipping
- Unique 8-digit identification number
- Links all your maritime records
- Required for CDC, COC, and all certifications
- Lifetime validity once registered

## Who Needs INDOS?

**Mandatory For**:
- All Indian seafarers
- Pre-sea training candidates
- Anyone applying for CDC
- Marine engineering students
- Nautical science students
- Ratings and officers alike

## INDOS Registration Process

### Step 1: Access E-Samudra Portal

**Website**: [esamudra.dgshipping.gov.in](https://esamudra.dgshipping.gov.in)

1. Click on "INDOS" section
2. Select "New Registration"
3. Choose your category (Deck/Engine/Cadet/Rating)

### Step 2: Fill Online Application

**Personal Details**:
- Full name (as per Class 10 marksheet)
- Date of birth
- Gender
- Father's name
- Mother's name
- Nationality (Indian)

**Contact Information**:
- Permanent address
- Communication address (can be same)
- Mobile number
- Email address

**Education Details**:
- Highest qualification
- Year of passing
- Institution name
- Percentage/CGPA

### Step 3: Upload Documents

**Required Documents (JPEG/PDF, max 200KB each)**:

| Document | Specifications |

| Signature | On white paper, black ink |
| Class 10 marksheet | Clear scan |
| ID proof | Aadhaar/Voter ID/Passport |
| Address proof | Aadhaar/utility bill |

## Documents Required - Detailed

### Mandatory Documents

**1. Passport Size Photograph**
- White background
- Face clearly visible
- Recent (within 6 months)
- 3.5 x 4.5 cm preferred
- No spectacles (if possible)

**2. Class 10 Marksheet**
- Clear, legible scan
- All details visible
- Board certification visible
- Name must match application

**3. ID Proof (Any One)**
- Aadhaar Card (preferred)
- Passport
- Voter ID
- PAN Card

**4. Address Proof (Any One)**
- Aadhaar Card
- Passport
- Utility bill (recent)
- Bank statement

### Additional Documents (If Applicable)

**For Existing Seafarers**:
- Old CDC copy
- COC copies
- Sea service certificates

**For Name Change**:
- Gazette notification
- Affidavit
- Marriage certificate (if applicable)

## Troubleshooting

### Application Rejected

**Common Reasons**:
1. Document quality issues
2. Information mismatch
3. Duplicate application
4. Incorrect category

**Solution**: Read rejection reason, rectify, reapply

### Status Shows Pending

**If pending for > 7 days**:
1. Login and check remarks
2. Contact DG Shipping helpdesk
3. Email: dgship-support@gov.in
4. Visit MMD if urgent

### Can't Login

**Solutions**:
1. Clear browser cache
2. Try different browser
3. Reset password
4. Contact helpdesk

### Documents Won't Upload

**Try**:
1. Reduce file size
2. Convert to PDF/JPEG
3. Use different scanner
4. Try during off-peak hours

## After INDOS Registration

### Next Steps

**For Pre-Sea Candidates**:
1. Apply for pre-sea course admission
2. Start STCW basic courses
3. Apply for CDC after course completion

**For Existing Seafarers**:
1. Link old CDC to INDOS
2. Update records if needed

### INDOS Updation

**Can Be Updated**:
- Address changes
- Contact information
- Qualification updates
- Photo renewal

**Cannot Be Changed**:

- Basic personal details (need supporting documents)

## FAQs

### How long is INDOS number valid?

INDOS number is valid for lifetime. It never expires and remains your unique identifier throughout your maritime career.

### Can I have two INDOS numbers?

No, only one INDOS number per person. If you have multiple registrations, contact DG Shipping to resolve and keep the correct one.

### I forgot my INDOS number. How to retrieve?

Login to E-Samudra with your registered email/mobile. Use "Forgot INDOS" option to retrieve. If email changed, contact MMD with ID proof.

### Can NRIs/PIOs get INDOS?

INDOS is for Indian citizens only. NRIs with Indian passport can register. PIOs and OCIs cannot get INDOS.

### Do I need INDOS for working on foreign flag ships?

Yes, INDOS is required for CDC which is mandatory even for foreign flag vessels as Indian seafarer. Your employer verifies through INDOS.
