# Pre-Sea Training Documents Checklist 2026: Complete Guide

Getting admission to pre-sea training requires extensive documentation. Missing even one document can delay your admission by weeks or months. This comprehensive checklist ensures you're fully prepared.

## Document Categories

### 1. Educational Documents

#### For GME/B.Tech Marine Engineering

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| Class 10 Marksheet | 5 original + 10 attested | CBSE/ICSE/State Board |
| Class 10 Certificate | 5 original + 10 attested | With date of birth |
| Class 12 Marksheet | 5 original + 10 attested | PCM subjects |
| Class 12 Certificate | 5 original + 10 attested | Pass certificate |
| Transfer Certificate | 2 original + 5 attested | From last institution |
| Migration Certificate | 2 original + 5 attested | If changing state |
| Character Certificate | 2 original + 5 attested | Recent (within 6 months) |

**Minimum Eligibility:**
- 60% aggregate in PCM (Physics, Chemistry, Mathematics)
- 50% individual in each subject
- Age: 17-25 years at time of admission

#### For DNS/B.Sc Nautical Science

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| Class 10 Marksheet | 5 original + 10 attested | CBSE/ICSE/State Board |
| Class 10 Certificate | 5 original + 10 attested | With date of birth |
| Class 12 Marksheet | 5 original + 10 attested | PCM subjects |
| Class 12 Certificate | 5 original + 10 attested | Pass certificate |
| Transfer Certificate | 2 original + 5 attested | From last institution |
| Migration Certificate | 2 original + 5 attested | If required |
| Character Certificate | 2 original + 5 attested | Recent |

**Minimum Eligibility:**
- 60% aggregate in PCM
- English as compulsory subject in 10th/12th
- Age: 17-25 years

#### For TME (Trainee Marine Engineering)

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| Class 10 Marksheet | 3 original + 5 attested | Pass certificate |
| Class 10 Certificate | 3 original + 5 attested | For age proof |
| Diploma Marksheets | 5 original + 10 attested | All semesters |
| Diploma Certificate | 3 original + 5 attested | Final certificate |
| AICTE Approval Letter | 2 copies | Of diploma institute |
| Transfer Certificate | 2 original | From polytechnic |

**Minimum Eligibility:**
- 3-year diploma in Mechanical/Marine/Naval Architecture
- 50% aggregate minimum
- Age: 17-25 years

#### For GP Rating

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| Class 10 Marksheet | 3 original + 5 attested | Minimum 40% |
| Class 10 Certificate | 3 original + 5 attested | Age proof |
| Character Certificate | 2 original | Recent |
| Transfer Certificate | 2 original | From school |

**Minimum Eligibility:**
- Class 10 pass with 40%
- English and Science subjects
- Age: 17.5-25 years

### 2. Identity Documents

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| Passport | 5 copies | All pages |
| Aadhar Card | 5 copies | Updated address |
| PAN Card | 3 copies | If available |
| Voter ID | 3 copies | If 18+ |

**Passport Requirements:**
- Valid for minimum 3 years from admission
- ECR (Emigration Check Required) category acceptable
- Apply early if not having passport

### 3. Medical Documents

| Document | Copies Required | Notes |
|----------|-----------------|-------|
| DG Shipping Medical | 2 original + 3 copies | From approved doctor |
| Eye Test Certificate | 2 original | With doctor's report |
| Blood Group Report | 2 copies | From recognized lab |
| Vaccination Records | 2 copies | If available |
| Physical Fitness Certificate | 2 original | From institute's doctor |

**Medical Fitness Standards:**

| Parameter | Deck | Engine |
|-----------|------|--------|
| Visual Acuity | 6/6 both eyes | 6/9 minimum |
| Color Vision | Normal (essential) | Normal (recommended) |
| Height | Minimum 157 cm | Minimum 157 cm |
| BMI | 18-30 | 18-30 |
| Hearing | Normal | Normal |

### 4. Photographs

| Type | Quantity | Specifications |
|------|----------|----------------|
| Passport size | 30 | White background, formal |
| Stamp size | 20 | White background |
| Full size | 10 | 4x6 inches, formal dress |

**Photo Specifications:**
- Recent (within 3 months)
- Matt finish preferred
- Without spectacles
- Ears visible
- No beard (preferably clean shaven)

### 5. Reservation Documents (If Applicable)

| Category | Documents Required |
|----------|-------------------|
| SC/ST | Caste certificate + Income certificate |
| OBC | OBC certificate (non-creamy layer) + Income |
| EWS | EWS certificate + Income certificate |
| PwD | Disability certificate (40%+) |
| Sports Quota | Achievement certificates + State/National |
| Defence | Relationship certificate + Discharge book |

**Important Notes:**
- Certificates must be recent (within 6 months)
- State-wise format varies
- NCL (Non-Creamy Layer) certificate for central quota

### 6. Financial Documents

| Document | Copies Required | Purpose |
|----------|-----------------|---------|
| Bank Statement | 3 copies | Fee payment proof |
| Income Certificate | 2 original | For scholarships |
| Education Loan Sanction | 2 copies | If applicable |

## Document Preparation Tips

### Attestation Guidelines

| Attesting Authority | Acceptable For |
|---------------------|----------------|
| Gazetted Officer | All documents |
| Notary | All documents |
| Bank Manager | Financial documents |
| Principal/College | Educational documents |

**Attestation Format:**
- "Certified true copy of original"
- Signature with designation
- Seal/stamp
- Date

### Document Organization

**Recommended Filing Order:**

| Section | Contents |
|---------|----------|
| Section 1 | Educational certificates (chronological) |
| Section 2 | Identity documents |
| Section 3 | Medical certificates |
| Section 4 | Photographs |
| Section 5 | Reservation documents |
| Section 6 | Miscellaneous |

**Tips:**
- Use clear file folders
- Index each document
- Keep multiple sets ready
- Original and copies separate

### Common Mistakes to Avoid

| Mistake | Solution |
|---------|----------|
| Missing attestation | Get all copies attested in advance |
| Expired certificates | Check validity dates |
| Name mismatch | Ensure name matches across documents |
| Poor photo quality | Use professional photo studio |
| Incomplete medical | Complete all tests as required |

## Conclusion

Document preparation is crucial for smooth pre-sea training admission. Start gathering documents at least 3-6 months before your expected admission date. Missing documents can delay your career start significantly.

Keep multiple copies, maintain digital backups, and verify all details for consistency across documents.
